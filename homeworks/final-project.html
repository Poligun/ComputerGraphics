Final Project

Source Code: https://github.com/Poligun/RayTracer

Currently Supported Features

  • Read / Write Bitmap
  • Multi Thread Rendering
  • Phong / Blinn Light
  • Recursive Reflection / Refraction (Using Fresnel Equation)
  • Load geometries from .obj files (Triangular faces only)
  • Vertex Normal Interpolation to Smooth Object: vn = (1 - u - v) * vn0 + u * vn1 + v * vn2 (See the teapot model)
  • Basic UV Texture Mapping

Future Improvements

  • Faster triangle intersection calculation
  • Full support of .obj and .mtl file format
  • Introduce CUDA Framework
  • Support more image formats
  • Read scenes from XML files
  • Animation
